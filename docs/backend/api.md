# API Endpoints

A API REST fornece os seguintes endpoints:

- `/api/projects/`: Para listar e criar projetos.
- `/api/projects/{id}/`: Para detalhar, atualizar e deletar um projeto.
- `/api/installments/`: Para listar e criar parcelas.
- `/api/installments/{id}/`: Para detalhar, atualizar e deletar uma parcela.

# Visão Geral do Projeto

Bem-vindo à documentação do projeto de Execução Orçamentária.

Este projeto é uma aplicação web para gerenciar e acompanhar a execução orçamentária de projetos.

## Tecnologias

- **Backend:** Django, Django Rest Framework
- **Frontend:** Vue.js
- **Banco de dados:** PostgreSQL

## Estrutura do Projeto

O projeto está dividido em duas partes principais:

- `backend`: Contém a aplicação Django com a API REST.
- `frontend`: Contém a aplicação Vue.js com a interface do usuário.

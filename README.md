# Sistema de Execução Orçamentária

Sistema para gestão e acompanhamento de execução orçamentária, desenvolvido com tecnologias atuais e containerizado com Docker.

## Tecnologias Utilizadas

### Frontend
- Vue.js 3
- Vuetify 3
- Chart.js
- Axios

### Backend
- Django 5
- Django REST Framework
- PostgreSQL

### Infraestrutura
- Docker
- Docker Compose
